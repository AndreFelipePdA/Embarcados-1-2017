1. Por que o Linux recebeu esse nome?
	O nome do criador do linux é Linus Torvalds
2. O que são daemons?
	No sistema operacional, o deamon é um programa executado como um processo
	em segundo plano em vez de estar sob o controle direto de um usuário interativo

3. O que é o shell?
		Um shell é um interpretador de linha de comando ou que fornece uma interface de
	usuário tradicional da linha de comando do Unix. 
		Os usuários direcionam a operação do computador digitando comandos como texto para
	um interpretador de linha de comando executar, ou criando scripts de texto de um ou mais
	ou mais desses comandos. 
	
4. Por que é importante evitar executar o terminal como super-usuário?
		Já que o super-usuário tem total controle sobre o sintema, pode manipular arquivos
	essenciais para seu funcionamento. 
		Um usuário sem experiencia pode corromper o sistema, sendo assim pode ser necessário
	ter que reinstalar o sistema novamente.
	
5. Qual botão do teclado completa o que o usuário escreve no terminal, de acordo com o contexto?
	Tecla "tab"

6. Quais botões do teclado apresentam instruções escritas anteriormente?
	Seta direcional para cima.
	
7. Apresente os respectivos comandos no terminal para:
  (a) Obter mais informações sobre um comando.

	$ man nome_do_comando
	ou	
	$ nome_do_comando --help
	ou	
	$ info nome_do_comando

  (b) Apresentar uma lista com os arquivos dentro de uma pasta.
	$ ls

  (c) Apresentar o caminho completo da pasta.
	$ pwd

  (d) Trocar de pasta.
	& cd nome_da_pasta

  (e) Criar uma pasta.
	& mkdir nome_da_pasta

  (f) Apagar arquivos definitivamente.
	$ rm nome_do_arquivo

  (g) Apagar pastas definitivamente.
	$ rm -r nome_da_pasta

  (h) Copiar arquivos.
	$ cp

  (i) Copiar pastas.
	$ cp - avr pasta1 pasta2

  (j) Mover arquivos.
	$ mv nome_do_arquivo parta_destino

  (k) Mover pastas.
	$ mv pasta1 pasta2 pasta_destino 

  (l) Renomear pastas.
	$ mv nome_antigo nome_novo

  (m) Apresentar o conteúdo de um arquivo.
	$ cat arquivo

  (n) Apresentar o tipo de um arquivo.
	$ file arquivo 

  (o) Limpar a tela do terminal.
	ctrl+L
	ou
	$ clear

  (p) Encontrar ocorrências de palavras-chave em um arquivo-texto.
	$ grep -rnw diretorio -e texto

  (q) Ordenar informações em um arquivo-texto.
	$ sort nome_do_arquivo
	
  (r) Substituir ocorrências de palavras-chave em um arquivo-texto.
	$ sed -i s/original/novo/g nome_do_arquivo

  (s) Conferir se dois arquivos são iguais.
	$ cmp -- silent antigo novo || echo "aquivos sao diferentes" 
	
  (t) Escrever algo na tela.
	$ echo "fase"
